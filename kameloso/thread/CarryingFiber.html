    Class CarryingFiber

    A core.thread.Fiber carrying a payload of type T.

    class CarryingFiber(T) ;

    Used interchangeably with core.thread.Fiber, but allows for casting to true CarryingFiber!T-ness to access the payload member.

    Constructors

    NameDescription
    this (fn, args) Constructor function merely taking a function/delegate pointer, to call when invoking this core.thread.Fiber (via .call()).
    this (payload, fn, args) Constructor function taking a T payload to assign to its own internal this.payload, as well as a function/delegate pointer to call when invoking this core.thread.Fiber (via .call()).

    Fields

    NameTypeDescription
    payload TEmbedded payload value in this core.thread.Fiber; what distinguishes it from normal ones.

    Methods

    NameDescription
    resetPayload () Resets the payload to its initial value.

    Example

    void dg()
{
    CarryingFiber!bool fiber = cast(CarryingFiber!bool)(Fiber.getThis);
    assert(fiber !is null);  // Correct cast
    assert(fiber.payload);
}

auto fiber = new CarryingFiber!bool(true, &dg, 32768);
