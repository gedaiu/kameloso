    Module kameloso.irccolours

    Functions related to IRC colouring and formatting, mapping it to ANSI terminal such, stripping it, etc.

    IRC colours are not in the standard as such, but there is a de-facto standard based on the mIRC coluring syntax of \3fg,bg...\3, where '\3' is byte 3, fg is a foreground colour number (of IRCColour) and bg is a similar background colour number.

    Example

    immutable nameInColour = "kameloso".ircColour(IRCColour.red);
immutable nameInHashedColour = "kameloso".ircColouByHash;
immutable nameInBold = "kameloso".ircBold;

    Functions

    NameDescription
    ircBold(word) Returns the passed string wrapped inbetween IRC bold control characters.
    ircColour(line, fg, bg) Colour-codes the passed string with mIRC colouring, foreground and background. Direct overload that leverages the output range version to colour an internal std.array.Appender, and returns the resulting string.
    ircColour(fg, bg) Returns a mIRC colour code for the passed foreground and background colour. Overload that doesn't take a string to tint, only the IRCColours to produce a colour code from.
    ircColourByHash(word) Returns the passed string coloured with an IRC colour depending on the hash of the string, making for good random nick colours in IRC messages.
    ircColourInto(line, sink, fg, bg) Colour-codes the passed string with mIRC colouring, foreground and background. Takes an output range sink and writes to it instead of allocating a new string.
    ircItalics(word) Returns the passed string wrapped inbetween IRC italics control characters.
    ircReset() Returns an IRC formatting reset token.
    ircUnderlined(word) Returns the passed string wrapped inbetween IRC underlined control characters.
    mapColours(line, fgReset, bgReset) Maps mIRC effect colour tokens to terminal ones.
    mapColours(line, fgReset, bgReset) Maps mIRC effect colour tokens to terminal ones.
    mapEffects(origLine, fgBase, bgBase) Maps mIRC effect tokens (colour, bold, italics, underlined) to terminal ones.
    mapEffects(origLine, fgBase, bgBase) Maps mIRC effect tokens (colour, bold, italics, underlined) to terminal ones.
    stripColours(line) Removes IRC colouring from a passed string.
    stripEffects(line) Removes all form of mIRC formatting (colours, bold, italics, underlined) from a string.
