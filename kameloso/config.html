    Module kameloso.config

    Functionality related to configuration; verifying it, correcting it, and reading it from/writing it to disk.

    Functions

    NameDescription
    applyDefaults(client, server) Completes a client's member fields with values needed to connect.
    complainAboutIncompleteConfiguration() Displays an error on how to complete a minimal configuration file.
    configurationText(configFile) Reads a configuration file into a string.
    readConfigInto(configFile, missingEntries, invalidEntries, things) Reads a configuration file and applies the settings therein to passed objects.
    writeConfigurationFile(instance, filename) Write all settings to the configuration filename passed.
    writeToDisk(filename, configurationText, banner) Saves the passed configuration text to disk, with the given filename.

    NameDescription
    ConfigurationFileReadFailureException Exception, to be thrown when the specified configuration file could not be read, for whatever reason.
