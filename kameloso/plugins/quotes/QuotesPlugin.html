    Class QuotesPlugin

    The Quotes plugin provides the ability to save and replay user quotes.

    class QuotesPlugin
      : IRCPlugin ;

    These are not currently automatically replayed, such as when a user joins, but can rather be actively queried by use of the quote verb.

    It was historically part of ChatbotPlugin.
