    Module kameloso.plugins.admin

    The Admin plugin features bot commands which help with debugging the current state of the running bot, like printing the current list of users, the current channels, the raw incoming strings from the server, and some other things along the same line.

    It also offers some less debug-y, more administrative functions, like adding and removing homes on-the-fly, whitelisting or de-whitelisting account names, adding/removing from the operator list, joining or leaving channels, and such.

    See the GitHub wiki for more information about available commands:
    - https://github.com/zorael/kameloso/wiki/Current-plugins#admin
