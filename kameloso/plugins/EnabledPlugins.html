    Alias EnabledPlugins

    List of enabled plugins. Add and remove to enable and disable.

    alias EnabledPlugins ;

    Due to use of tryImportMixin above only files actually present will have been imported.

    Note that dub will still compile any files in the plugins directory! To completely omit a plugin you will either have to compile the bot manually, delete the source file(s) in question, or add an __EOF__ at the top of them. Everything below a line with that text is skipped. Make sure it's above the module declaration.
