    Function immediate

    Immediately sends text to the server, verbatim. Skips all queues.

    void immediate (
      ref IRCPluginState state,
      const(string) line
    );

    This is used to send messages of types for which there exist no helper functions, and where they must be sent at once.

    See Also

    raw

    Parameters

    NameDescription
    state Reference to the current plugin's kameloso.plugins.common.IRCPluginState, via which to send messages to the server.
    line Raw IRC string to send to the server.

    Example

    import kameloso.thread : ThreadMessage;
import lu.conv : Enum;
import std.meta : AliasSeq;

IRCPluginState state;
state.mainThread = thisTid;

immediate(state, "test");

try
{
    receiveOnly!(AliasSeq!(ThreadMessage.Immediateline, string));
}
catch (Exception e)
{
    assert(0, "Receiving an `immediateline` failed.");
}
