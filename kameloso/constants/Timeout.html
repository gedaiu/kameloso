    Enum Timeout

    Various timeouts in seconds.

    enum Timeout : int { ... }

    Enum members

    NameDescription
    connectionLost How long to keep trying to read from the sever when not receiving anything at all before the connection is considered lost.
    initialPeriodical How long to wait before calling plugins' periodical for the first time.

    Since it is meant for maintenance and cleanup tasks we can hold on a while before calling it the first time.

    readErrorGracePeriod How long to wait after encountering an error when reading from the server, before trying anew.

    Not having a small delay could cause it to spam the screen with errors as fast as it can.

    retry The amount of seconds to wait before retrying after a failed connection attempt.
    whoisRetry How long to wait before allowing to re-issue a WHOIS query for a user.

    This is merely to stop us from spamming queries for the same person without hysteresis.
